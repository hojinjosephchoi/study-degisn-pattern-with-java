# Strategy Pattern

## 학습목표
- Interface 개념
- 델리게이트 개념
- 전략패턴 개념

## Interface
- 기능에 대한 __선언과 구현 분리__
- 기능을 사용 __통로__

## Delegate
- __위임__하다
- 떠 넘기는 역할
- 특정 객체의 기능을 사용하기 위해 다른 객체의 기능을 호출한다.


## Strategy Pattern이란
- 여러 알고리즘을 하나의 __추상적인 접근점__ (인터페이스) 을 만들어 접근 점에서 서로 __교환 가능__하도록 하는 패턴
